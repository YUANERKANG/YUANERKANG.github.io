# YUANERKANG.github.io

这是员尔康的 GitHub Pages 个人网站，使用纯静态 HTML、CSS 和 JavaScript，不需要构建工具。

## 如何使用

1. 如需更新内容，修改 `index.html` 里的项目介绍、技能方向和联系方式。
2. 把本文件夹推送到 GitHub 仓库。
3. 在 GitHub 仓库中打开 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/root` 目录，然后保存。

网站地址：

```text
https://YUANERKANG.github.io/
```
